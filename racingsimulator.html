Autonomous Racing AI Development

Project Overview

This autonomous control system was developed using a custom simulation environment I created specifically for training AI agents in vehicle dynamics and racing line optimization.

Simulation Environment

  • Custom physics engine
  • Configurable track generation
  • Sensor data emulation

AI Framework

  • Python-based training pipeline
  • PyTorch integration
  • OpenAI Gym interface

Development Journey

Phase 1: Supervised Learning Approach

Initial implementation used supervised learning with:

  • Regression models for continuous control outputs
  • Random Forest classifiers for discrete action selection

While these produced functional results, they demonstrated limitations in adaptability and required extensive labeled training data.

Phase 2: Reinforcement Learning Transition

Recognizing the constraints of supervised methods, I transitioned to reinforcement learning:

  • Initial attempt with Deep DQN showed no convergence due to simulation environment issues
  • Second attempt using PPO in Racing Gym environment confirmed algorithm viability
  • Simulation debugging resolved physics and reward function issues
  • PPO implementation (Proximal Policy Optimization) proved successful
