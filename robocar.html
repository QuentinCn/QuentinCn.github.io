Robocar

I started this project with a friend, then my school got interested in it and asked us to make it a national module.

The project is splitted in mulitple parts

The main parts of this project are:

  • AIs

    • RacingSimulator
  • Simulation
  • Hardware
  • Embedded software
  • 3D printed parts
I will only present you the parts i did

MaskGenerator is an AI which generate a mask of lines from an image of a road. It is entirely train from the simulation
